# Grammars Extension

[![Lint Check](https://github.com/tokenspan/grammars/actions/workflows/lint.yml/badge.svg?branch=main)](https://github.com/tokenspan/grammars/actions/workflows/lint.yml)
[![Build And Upload Extension](https://github.com/tokenspan/grammars/actions/workflows/build-zip.yml/badge.svg)](https://github.com/tokenspan/grammars/actions/workflows/build-zip.yml)

This is a Chrome extension that provides grammar correction and spelling suggestions for various websites.

## Features

- Grammar correction for various websites
- Spelling suggestions for various websites
- Customizable settings for grammar correction and spelling suggestions
